#### 使用说明

1.  根据本地环境，先创建数据库、导入resources/sql 里面的语句，更改pom.xml文件里面的数据库等链接参数
2.  运行IcoolleApplication类,启动项目
3.  访问http://localhost:8888/home,选择代码生成，可以根据数据库生成通用代码
4.  访问http://localhost:8888/doc.html，此地址是接口文档地址
